# README

Отборочный контест LLM Scaling Week от ШАД.

## Задачи

- Предсказание ошибок
- Лаборатория биокибернетики

Зависимости определены [pyproject.toml].

Для управления зависимости рекомендую использовать `uv`:

```bash
uv run jupyter notebook
```
